# STL 輪郭切断スライサー「CutStack」v2.0

**STL形式の3Dモデルを指定した板厚でスライスし、レーザーカッター用のSVG／PDFを出力するWebアプリケーションです。**
**Keywords:** STL, Laser Cutting, Digital Fabrication, STEAM Education

## 概要

CutStackは、STL形式の3Dモデルファイルを一定間隔でスライスし、各断面の輪郭線をSVGとして板材に配置して出力するツールです。レーザーカッターで板材を切り出し、順番に積み重ねることで3Dモデルの立体形状を再現できます。

### 主な特徴

- STLファイルの読み込み（バイナリ・ASCII両対応）
- 断面輪郭の自動再構成・修復
- 板への自動配置（スペース最適化）
- SVG／PDF形式での出力（ベクター品質）
- 組み立て完成図の3Dプレビュー（ドラッグ回転・ズーム対応）
- シンプル版／フル機能版の2モード

## 教育利用

本ツールは以下の教育用途を想定しています：

- STEAM教育におけるデジタルファブリケーション
- 3Dモデルと2D展開の関係理解
- レーザーカッター実習教材

## 起動方法

5つのファイル（`index.html`、`simple.html`、`full.html`、`app.js`、`styles.css`）を同じフォルダに置き、`index.html` をブラウザで開いてください。

> **対応ブラウザ：** Chrome / Edge / Firefox / Safari（最新版推奨）  
> **インストール不要：** サーバー不要、ローカルで動作します

2. STLファイルを読み込む
3. スライス生成

## モード選択

起動時にモードを選択します。

| モード | 対象 | 特徴 |
|---|---|---|
| **シンプル版** | はじめての方・手軽に使いたい方 | 板厚・板幅・板高さのみ設定。ボタン1つでスライス生成 |
| **フル機能版** | 詳細設定・診断が必要な方 | 全パラメータ設定、スライス結果の詳細診断機能付き |

各版の画面右上または左下のリンクからいつでも切り替えられます。

## シンプル版 — 操作手順

### 1. STLファイルを読み込む

「STLファイル」ボタンをクリックし、対象のSTLファイルを選択します。

### 2. 基本設定を入力する

| 設定項目 | 説明 | 初期値 |
|---|---|---|
| **板厚 (mm)** | 使用する板材の厚さ。スライス間隔になります | 3 mm |
| **板幅 (mm)** | レーザーカッターで使用する板材の幅 | 450 mm |
| **板高さ (mm)** | 板材の高さ（奥行き） | 300 mm |

### 3. スライス生成

「**スライス生成**」ボタンをクリックします。処理が完了すると配置プレビューが表示されます。

### 4. 結果を確認する

- ステータスボックスに部品数・板枚数・閉曲線数・開放輪郭数が表示されます
- 「前の板」「次の板」ボタンで複数板がある場合に切り替えられます

### 5. 出力する

| ボタン | 動作 |
|---|---|
| **全板SVG保存** | 全板分のSVGファイルを個別にダウンロード |
| **全板PDF保存** | 全板を1つのPDFファイルにまとめてダウンロード（ベクター品質） |

## フル機能版 — 操作手順

### 設定パラメータ

| 設定項目 | 説明 | 初期値 |
|---|---|---|
| **板厚 (mm)** | 板材の厚さ・スライス間隔 | 3 mm |
| **部品間隔 (mm)** | 板上での部品同士の余白 | 2 mm |
| **板幅 (mm)** | 板材の幅 | 450 mm |
| **板高さ (mm)** | 板材の高さ | 300 mm |
| **接続許容値 (mm)** | 断面輪郭の端点を同一点とみなす距離の上限 | 1.8 mm |
| **修復許容値 (mm)** | 開放輪郭を自動修復する際のギャップ許容値 | 1.0 mm |
| **最小線分長 (mm)** | この長さ未満の線分は除外する | 0.1 mm |
| **最小開放輪郭長 (mm)** | この長さ未満の開放輪郭は除外する | 1.5 mm |

### 表示オプション

| チェックボックス | 説明 |
|---|---|
| **部品番号表示** | 配置プレビューに部品番号（S01〜）を表示 |
| **次部品の輪郭線表示** | 現部品に次番号部品の輪郭を青で重ねて表示（組み立て時の位置合わせ用） |
| **修復前後比較表示** | 診断パネルに修復前・修復後の輪郭を並べて表示 |
| **ペア候補表示** | 分岐ノードでの採用ペア情報を診断テーブルに表示 |

### ナビゲーションボタン

| ボタン | 動作 |
|---|---|
| **前の板 / 次の板** | 複数板がある場合に板を切り替え |
| **◀ 前のスライス / 次のスライス ▶** | 診断パネルで確認するスライスを切り替え |

### ズーム操作

| 青 実線 | 次部品の輪郭（組み立て位置合わせ用） |

## スライス結果診断（フル機能版）

### 診断ボックス

スライス処理後、輪郭品質の概要を表示します。

- **輪郭品質は概ね良好です** — 開放輪郭がほぼない状態
- **開放輪郭が残っています** — 接続許容値や修復許容値の引き上げを検討してください

### スライス情報

| 項目 | 説明 |
|---|---|
| **対象スライス** | 現在表示中のスライス番号 |
| **Z位置** | スライス面のZ座標（mm） |
| **閉曲線数** | 正常に閉じた輪郭の数 |
| **開放輪郭数** | 閉じられなかった輪郭の数（0が理想） |
| **修復済み** | 自動修復された輪郭の数 |
| **次部品** | 次番号部品の有無 |

### 修復前後比較

スライスごとに輪郭の修復前（赤）と修復後（赤）を並べて表示します。

### 輪郭の閉曲線チェックテーブル

| 列 | 説明 |
|---|---|
| **輪郭** | 輪郭番号（P01〜） |
| **状態** | 閉（緑）/ 開（赤） |
| **修復** | 閉鎖・結合・最近傍・— |
| **点数** | 輪郭を構成する頂点数 |
| **端点ギャップ** | 開放時の端点間距離（mm） |

## PDF出力仕様

- 全板を1ファイルにまとめて出力
- **ベクター品質**（パス情報を直接PDFに埋め込み、拡大しても劣化なし）
- ページサイズ：板幅 × 板高さ（mm → pt換算）
- フォント：Helvetica（PDF組み込みフォント）

## 輪郭修復の仕組み

STLメッシュの浮動小数点誤差や欠損により、断面輪郭が閉じない場合があります。CutStackは以下の順で自動修復を行います。

1. **パス1：端点閉鎖** — 単一輪郭の端点が接続許容値以内なら閉じる
2. **パス2：スコアマージ** — 開放輪郭同士を端点の距離・方向スコアで最良ペアリングして結合
3. **パス3：広域マージ** — 接続許容値の2倍の範囲で再試行
4. **パス4：広域端点閉鎖** — 広い許容値で単一輪郭を再度閉鎖
5. **パス5：最近傍グラフ結合** — 残る開放輪郭を距離無制限で最近傍端点から強制結合（STLの大きな欠損にも対応）

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| 開放輪郭が多い | 接続許容値を 2.0〜3.0 mm に上げてみてください |
| 輪郭が崩れている | 接続許容値を下げてみてください（大きすぎると近傍の別点が誤結合されます） |
| 部品が板に収まらない | 板幅・板高さを大きくするか、板厚を増やして部品数を減らしてください |
| STL読み込みに失敗する | ASCII STL・バイナリ STL 両対応ですが、ファイルが破損していないか確認してください |
| PDFが開けない | Adobe Acrobat Reader または最新ブラウザのPDFビューアで開いてください |
